# Карта файла COMETAL 2.0 18.06

Источник: https://www.figma.com/design/wt7rhRt3u2xq4t2ajUZ90R/COMETAL--2.0-18.06?m=auto&t=L9FxkjNzn3D3mWHi-6

Основание классификации: список страниц и структурные признаки, полученные через Figma MCP: наличие page divider, количество дочерних узлов, секций, компонентов и component sets. Статусы в названиях интерпретированы так: `✅` — актуальная страница, `🟨` — страница в работе / частично актуальная, `Архив/Черновик` — неактуальная архивная зона.
