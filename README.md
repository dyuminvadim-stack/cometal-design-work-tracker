# Process Tracker Kit

Локальное веб-приложение для упаковки рабочего плана в визуальный трекер. По умолчанию внутри лежит COMETAL Redesign Navigator, но теперь приложение можно переиспользовать для любого процесса через импорт JSON-плана.

## Запуск

```bash
npm install
npm run dev
```

После запуска откройте URL, который покажет Vite, обычно:

```text
http://localhost:5173
```

## Хранение данных

Активный трекер, состояние задач и прикрепленные файлы результатов хранятся в `localStorage` браузера. Кнопка `Export` скачивает ZIP-архив с `pipeline_state.json` и всеми загруженными файлами результатов.

## Импорт нового плана

1. Подготовьте JSON по формату из `tracker_intake.md`.
2. Нажмите `Import` в приложении.
3. Выберите JSON-файл.
4. Работайте с задачами, прикрепляйте результаты и экспортируйте ZIP.

Стартовый пример лежит в `src/tracker_plan_template.json`.

Кнопка `Reset` сбрасывает прогресс текущего трекера. Кнопка `Default` возвращает исходный COMETAL-трекер.

## UI

Интерфейс собран на React, TypeScript и Bootstrap 5. Дефолтный COMETAL-пайплайн лежит в `src/pipeline.json`.
